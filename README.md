# study-alone

1. module-prac 
 - 여러 파일에 걸쳐 재사용 되는 함수나 변수를 모듈로 만들어서 편리하게 사용하기 위한 목적입니다.
 - 간단하게 구현 후 동작 확인을 했습니다.
  
2. global-prac
 - 노드의 내장 객체인 global.
 - 전역 객체라는 점을 활용하여 파일 간에 간단한 데이터를 공유 하는 법 공부했습니다.
 
3. filename-dirname
 - 노드에서 파일 사이에 모듈 관계가 있는 경우 현재파일의 경로와 파일명을 알아 볼 수 있도록 console.log로 한번 찍어보았습니다.
